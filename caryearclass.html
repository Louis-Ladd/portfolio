Car Year Classifier

A Nerual Network Classifier that can determine the production year from images of cars from 1980 to 2020

Descriptions

This project was for a Data Science class I took, we covered the basics of data processing, analysis, and advanced computer learning algorithms such as k-means clustering, A*, Naive Bayes and Linear/Polynomial Regression. I also learned how to analyze data using correlation matrixies and data analysis/visulaization libraries such as matplotlib, pandas and scikit.
