Car Year Classifier

My final project for a Data Science class which involved using TensorFlow to create a neural network. The goal was to train the network to identify the year of cars in images. I enjoyed this project quite a bit and I was able to get a 70% accuracy on 15,000 images (70/30 train test split). The above image is a neuron activation heat map of the network showing what the network was looking at to make its decisions.
